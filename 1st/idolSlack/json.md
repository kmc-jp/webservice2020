## 概要
jsonの目的、文法を理解する。

## 目的
jsonをつかうと、変数のデータなどを格納することができる。便利。

## 文法
### 基本

```json
{
    "Key1":"Data1",
    "Key2":"Data2",
    ...
}
```

このようにmap(ハッシュ)型のデータを格納する。

```json
[
    "Data1",
    "Data2",
    ...
]
```

これでスライス(配列)のデーガを格納する。

簡単ですね！

### 発展

組み合わせて入れ子も出来る。

```json
{
    "Key1":"Data1",
    "Key2":[
        "Data2",
        "Data3"
        ...
    ],
    ...
}
```
